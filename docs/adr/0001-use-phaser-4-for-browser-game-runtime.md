# Use Phaser 4 for the Browser Game Runtime

Chicken Olympics is a real-time local browser match with simultaneous keyboard movement, timed chick peeks, overlap-based claiming, sprite animation, scene transitions, and later sound. We will use Phaser 4 with TypeScript as the core runtime, scaffolded as a Vite + TypeScript browser app with no React, and the first playable version's title screen, color selection, match, and podium ceremony will all be Phaser scenes rather than a React or DOM shell. Player chicken movement and chick claiming will use Phaser Arcade Physics for simple bounds and overlap checks, not Matter.js or custom collision code. Before building the full MVP, the first implementation slice must validate current Phaser 4 risk with a tiny playable spike covering keyboard input, overlap claiming, timers, and scene transitions; this accepts that `phaser.io` and `docs.phaser.io` were unreachable from this environment during evaluation, while npm and GitHub showed Phaser 4 as current and actively maintained.
